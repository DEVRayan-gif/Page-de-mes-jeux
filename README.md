# Gaming RAYAN

Ce projet est une page web de type galerie gaming présentant plusieurs jeux vidéo auxquels j’ai joué.  
Il a été réalisé en HTML, CSS et JavaScript.

---

## Description du site

Le site contient une page principale :

### Gaming RAYAN
- Présentation visuelle du thème gaming  
- Affichage de plusieurs jeux vidéo auxquels j’ai joué et que j’apprécie  
- Interface moderne et stylée  
- Mise en avant de différents univers de jeux  

---

## Jeux présentés

- Valorant  
- League of Legends  
- GTA V Online  
- CS:GO  
- Rainbow Six Siege  
- Rocket League  

---

## Technologies utilisées

- HTML5  
- CSS3   

---

## Objectif du projet

Ce projet a pour but de :
- améliorer mes compétences en développement web  
- apprendre à créer une interface visuelle attractive  
- travailler le design et l’UX  
- organiser une page de type galerie
